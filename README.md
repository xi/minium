minium - minimal vim keybindings for chromium

key | action
--- | -------------
h   | scroll left
j   | scroll down
k   | scroll up
l   | scroll right
gg  | scroll to top of the page
G   | scroll to bottom of the page

For a more complete set of vim keybindings for chromium see
[Vimium](https://github.com/philc/vimium).
